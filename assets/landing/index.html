Frequently Asked Questions

Everything you need to know before making your first API call.

Is there a free plan?
Yes. The BASIC plan is completely free with 50 AI calls and 300 total requests per month. No credit card required. Instant scoring endpoints don't count against the AI quota.
How fast are the instant endpoints?
The 8 heuristic scoring endpoints (score_tweet, score_linkedin_post, score_instagram, score_youtube_title, score_email_subject, score_multi, score_readability, analyze_headline) run in <5ms on the server. No AI call, no wait time.
How does the scoring system work?
Each scorer uses a heuristic model tuned for that platform. Scores are 0–100 with letter grades A–F. For example, the tweet scorer checks character length (optimal: 71–240), hashtag count (1–3 is ideal), emoji presence, power words, question hooks, and readability. Every scorer returns suggestions explaining how to improve.
What does score_multi do?
It scores your text against all 5 platforms (tweet, LinkedIn, Instagram, YouTube, email) in a single API call. Useful for deciding which platform a particular piece of copy is best suited for before you start adapting or rewriting it.
Can I integrate ContentForge with Zapier, Make, or n8n?
Yes. All endpoints accept standard JSON over HTTPS and return consistent JSON responses. The RapidAPI gateway handles auth via the X-RapidAPI-Key header. Works with any automation tool that supports HTTP requests.
What AI model powers the AI endpoints?
The AI endpoints use Google Gemini 2.0 Flash via the Gemini API. Responses are typically under 3 seconds. The instant scoring endpoints use zero AI — they are purely algorithmic and always available.
Is the OpenAPI spec available?
Yes. The full OpenAPI 3.0 spec is available in the GitHub repository at deploy/openapi.json. The RapidAPI listing also provides interactive docs for every endpoint.
